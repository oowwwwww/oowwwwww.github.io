线性回归

通过已知的数据来预测一个新值。

h（假设函数）的表示：$h_θ(x) = θ_0 + θ_1x$，为了预测一个关于x的y的方程。

代价函数（cost function)[平方差代价函数]

不同的θ0和θ1对应着不同的代价函数，预测得到的值也是不同的，为了使预测的值能够更准确，就要找到合适的θ0和θ1使得代价函数最小，即使h函数和数据更加拟合。

其中一个方法就是采用平方误差和，将每一个h函数对应的值和真实值的差取平方和相加再取平均值，当取得的θ0和θ1能使平方差代价最小的时候就是拟合效果最好的时候。

梯度下降

梯度下降是一个用来求函数最小值的算法，可以采用梯度下降算法来求代价函数的最小值。

重复直到收敛：$θ_j := θ_j - \alpha\frac{\partial}{\partial j}J(\theta _0, \theta _1)$ ($for\ j = 0\ and\ j = 1$) 其中$\alpha$是learning rate，相当于“下山的步子”，当学习率越大变化就越快，但也有可能错过最低点，当学习率越小变化就越慢，但是一般会找到损失最低点。

如图所示，当初始点在右侧时，当切线斜率为正时，$\theta _1$ 会减小，不断地更新直到切线等于零时变为最小。当初始点在左侧时，切线斜率为负，$\theta _1$会变大直到收敛为止。并且可以从图中看到，当接近局部最低的时候，梯度下降会自动的采用更小的步子（斜率在变小），因此不需要再另外减小学习率。

线性回归的梯度下降

通常线性回归的代价函数通常是凸函数，它没有局部最优解，只有一个全局最优解，所以它一定能够收敛到最低点。梯度下降又叫“Batch”梯度下降，因为每一次都是$\sum _{i=1}^{m} $计算m个数据。
