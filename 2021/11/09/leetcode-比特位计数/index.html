leetcode-比特位计数

1. Brian Kernighan算法

首先学习一下 Brian Kernighan算法

Brian Kernighan算法：对任何一个数 n nn，n & ( n − 1 ) n&(n-1)n&(n−1)的结果是n nn的比特位最右端的1变为0的结果。

题目描述：给你一个整数 n ，对于 0 <= i <= n 中的每个 i ，计算其二进制表示中 1 的个数 ，返回一个长度为 n + 1 的数组 ans 作为答案。

📖解题思路：就是Brian Kernighan算法，不断地遍历，直至为0，计数遍历的次数就能得到对应的1的个数了。

复杂度分析

  • 时间复杂度：O(nlogn)。需要对从 0 到 n 的每个整数使用计算「一比特数」，对于每个整数计算「一比特数」的时间都不会超过 O(logn)。

  • 空间复杂度：O(1)。除了返回的数组以外，空间复杂度为常数。

2. 动态规划

📖解题思路：可以将所有的数分为奇偶来考虑，奇数包含1的个数一定比前面那个偶数多一个1，就是最低位的1，对于偶数，偶数中的1的个数一定和它除以2之后的那个数一样多，因为除以2就是向右移1位，而偶数的最低位为0，所以1的个数不变，因此可以通过遍历得到1的个数了。
